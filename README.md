# GOST 代理可视化工具

一个基于 Electron 和 Vite 构建的 GOST 代理可视化管理工具，用于方便地创建、管理和监控 GOST 代理任务。

## 功能特点

- **多任务管理**：支持同时创建和管理多个代理任务
- **两种模式**：支持端口映射模式和代理模式
- **实时监控**：实时显示任务运行状态和日志
- **系统托盘**：最小化到系统托盘，支持从托盘快速操作
- **主题切换**：支持浅色/深色主题切换
- **任务持久化**：任务配置会自动保存，重启应用后保持
- **批量操作**：支持一键启动/停止所有任务

## 技术栈

- **前端**：原生 JavaScript + CSS
- **构建工具**：Vite
- **桌面应用**：Electron
- **代理核心**：GOST

## 安装和运行

### 前提条件

- Node.js 14.0 或更高版本
- npm 6.0 或更高版本

### 安装依赖

```bash
npm install
```

### 开发模式运行

```bash
npm run electron:dev
```

### 构建应用

```bash
npm run electron:build
```

构建完成后，可执行文件将位于 `dist` 目录中。

## 使用说明

### 创建新任务

1. 在应用主界面的 "配置" 选项卡中填写任务参数：
   - **任务名称**：自定义任务名称
   - **本地协议**：选择本地代理协议（http/https/socks5/tcp）
   - **本地端口**：设置本地监听端口
   - **远程协议**：（可选）选择远程代理协议
   - **远程地址**：（可选）设置远程代理服务器地址
   - **远程端口**：（可选）设置远程代理服务器端口
   - **模式**：选择 "代理" 或 "端口映射" 模式
   - **目标地址**：（端口映射模式）设置映射目标地址
   - **目标端口**：（端口映射模式）设置映射目标端口
   - **网络接口**：（可选）指定网络接口
   - **调试模式**：（可选）启用调试模式

2. 点击 "创建任务" 按钮启动任务

### 管理任务

- **启动任务**：点击任务卡片上的 "启动" 按钮
- **停止任务**：点击任务卡片上的 "停止" 按钮
- **删除任务**：点击任务卡片上的 "删除" 按钮
- **查看日志**：点击任务卡片上的 "显示/隐藏" 按钮查看任务日志

### 批量操作

- **启动所有任务**：点击界面顶部的 "启动所有任务" 按钮
- **停止所有任务**：点击界面顶部的 "停止所有任务" 按钮

### 系统托盘操作

- **打开窗口**：左键点击系统托盘图标
- **停止服务**：右键点击系统托盘图标，选择 "停止服务"
- **退出**：右键点击系统托盘图标，选择 "退出"

## 项目结构

│   ├── app-icon.png     # 应用图标
│   └── vite.svg         # Vite 图标

│   ├── counter.js       # 计数器模块

├── main.js              # Electron 主进程
├── package-lock.json    # npm 依赖锁定文件

## 注意事项

1. 本工具基于 GOST 实现代理功能，需要确保 `gost.exe` 文件存在于项目根目录
2. 运行代理服务可能需要管理员权限，特别是在使用低端口号时
3. 请合理使用代理服务，遵守相关法律法规
4. 任务配置会保存在应用数据目录的 `tasks.json` 文件中
